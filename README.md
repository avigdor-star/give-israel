# Give Israel

The most comprehensive directory of Israeli charities. Browse by category, search by name, and donate directly.

- **45+ verified charities** across 15 categories
- **Real-time search** and category filtering
- **Community submissions** — anyone can suggest a charity
- **Admin dashboard** for reviewing and publishing suggestions
